# Angela Guilherme — Full Stack Developer Portfolio

A modern, Apple-inspired portfolio with a cinematic interface, smooth scrolling, and a demo e-commerce experience.
Designed with clarity, performance, and refined UI principles.

---

## <i data-lucide="layout-dashboard"></i> Features

### <i data-lucide="palette"></i> Design & UX

* Minimal Apple-style layout and typography
* Glassmorphism with controlled gradients
* Smooth scrolling with section reveal animations
* Light / Dark mode toggle
* Fully responsive design

### <i data-lucide="folder"></i> Portfolio System

* Dynamic GitHub projects integration
* Auto-generated project cards
* Search and filtering functionality
* Structured grid layout inspired by modern design systems

### <i data-lucide="shopping-cart"></i> Demo E-Commerce

* Client-side shopping cart
* Add to cart interaction
* Slide-in cart panel
* Checkout modal interface

### <i data-lucide="credit-card"></i> Payment UI (Demo)

* Credit card form interface
* Visual support for:

  * Visa
  * Mastercard
  * American Express
* No authentication or backend required (demo flow)

---

## <i data-lucide="wrench"></i> Tech Stack

This portfolio includes:

* A cookie-clicker game
* A personal library
* An event booking website
* A personal finance guidance site
* A private organisation management app
  
* A simulated e-commerce experience:

* Complete and ready online store
* Product selection, wishlist, add product to basket
* Cart and checkout are frontend-only
* Payments are not processed
* No real transactions occur
